--- Appointment Scheduling System ---

-About-
Purpose of this application is to run an appointment scheduling system given business constraints, such as limited hours and checking for overlap, as well as storing
client and server information. It is compatible with French and English as well as all time zones. The application and business hours adjust to the users local time zone.
